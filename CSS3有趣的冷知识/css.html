这是测试calc()函数：css样式计算。

代码：
height: calc(100% - 500px);

效果与作用： 查看页面的设备不同（或窗口大小不同），高度不同，它也将随着计算变化,该元素的高度始终为父元素减去500px。
注意： 若将元素position属性设置为relative或static（默认）将会是calc计算失效，absolute, fixed可以。 使用calc(100% + 100px);这样的形式效果最好，还可以与不同设备进行自适应。awsome！
这里也用到了calc()函数计算样式中top属性。 测试模糊文字。

原理： text-shadow属性配合前景色透明实现滤镜效果。
代码：
top: calc(100% - 350px);
height: 350px;
color: transparent;
text-shadow: 0 0 2px #ff00ff;

测试background-clip属性

css3中的background-clip属性用于剪裁css元素。

下面是一段彩色文字：

代码如下：

.demo.demo5 .wordColor {
font-size: 40px;
font-family: STXingKai;
color: transparent;/*将文字设置颜色透明*/
background: linear-gradient(to right, red, orange, yellow, green, blue,indigo, violet, transparent);/*线性渐变*/
-webkit-background-clip: text; /*剪裁元素到文字（text）*/
/*-webkit-background-clip与background属性顺序决定了是否clip生效，必须background设置渐变再有clip到text,否则将无用。*/
}

这是利用background-clip与transparent属性制作的一段彩色文字。这里有个坑：就是必须要设置背景渐变，也就是要有背景色，后才能使-webkit-background-clip属性生效，也就是产生彩色文字效果。
